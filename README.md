# 캠핑캐치 (CampCatch) — 오토캠핑 랭킹 대시보드

김포 집 기준 **약 3시간 이내** 지역의 **오토캠핑장**을, 조회일 기준으로
**후기 좋은 순 + 자리 있는 곳 우선**으로 줄 세우고 **계절 추천**과 **가격**까지
한 화면에서 보여주는 프로토타입 대시보드입니다.

> ⚠️ **데모 데이터**: 표시되는 캠핑장명·후기·가격·자리 현황은 모두 레이아웃/동작
> 확인용 예시값이며 실제 데이터가 아닙니다. 실서비스화하려면 `js/data.js`를
> 실데이터로 교체하고, 후기·가격·자리 수집은 각 예약 플랫폼의 약관을 검토해야 합니다.

## 기능

- **후기 기반 자리 랭킹** — 조회일 기준 후기 좋은 순 + 그날 자리 있는 곳 우선
- **선호 우선 제안** — ☆를 눌러 지정한 '선호' 캠핑장은 랭킹 최상단에 고정 노출
  (브라우저 localStorage에 저장되어 재방문 시 유지)
- **가족캠핑 필터** — 아이 동반 가족캠핑에 맞지 않는 곳(노키즈존·커플 전용)은 목록에서 제외
- **계절 추천** — 날짜의 계절을 판별해 여름(계곡·물놀이·그늘) / 겨울(온수·난방·설경)에
  맞는 곳을 상단 배너·큐레이션·랭킹 가점에 반영
- **가격 표시** — 카드·상세에 요금 표시(조회일에 따라 주중/주말/성수기 자동 판별)
- **집 기준 이동시간** — 김포 기준 예상 소요시간 표시, 지역 필터
- **취소표 알림(보조)** — 마감된 곳에만 붙는 알림 신청 버튼
- 라이트/다크 테마 자동 대응, 모바일 우선 반응형

## 지역 / 조건 (기본값)

- 지역: 파주 · 연천 · 포천 · 강원 (김포 집 기준 약 3시간 이내)
- 유형: 오토캠핑 고정
- 대상: 가족캠핑(아이 동반) — 노키즈존·커플 전용은 자동 제외

## 랭킹 로직

```
순위점수 = 후기점수(별점×10 + 리뷰수 보정)
         + 자리가용(조회일 예약가능 시 +25)
         + 계절적합(시즌 태그 매칭 × 4)
         − 이동시간 페널티(시간 × 3)
```

계절별 "가면 좋은 곳" 정의와 캠핑장 태그는 `js/data.js`의 `CC.SEASON`, `CC.SITES.tags`에 있습니다.

## 파일 구조

```
camping-dashboard/
├─ index.html        # 화면 구조
├─ css/styles.css    # 디자인 토큰(라이트/다크) + 컴포넌트
└─ js/
   ├─ data.js        # 캠핑장·계절·후기 데이터 (여기만 교체하면 실데이터 반영)
   ├─ logic.js       # 계절/요금/자리/랭킹 등 순수 로직
   └─ app.js         # 상태·화면 렌더링·이벤트
```

의존성 없음(바닐라 JS). 스크립트는 전역 네임스페이스 `window.CC`를 공유합니다.

## 실행

빌드 과정이 없습니다. 다음 중 하나로 확인하세요.

- `index.html`을 브라우저로 직접 열기
- 또는 로컬 서버:
  ```bash
  cd camping-dashboard
  python3 -m http.server 8000   # http://localhost:8000
  ```

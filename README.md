# 캠핑캐치 (CampCatch) — 김포 3시간 오토캠핑 대시보드

김포 집 기준 **약 3시간 이내** 지역의 **오토캠핑장**을, 조회일의 **계절**과 **거리**로
추천 순위를 매겨 보여주는 대시보드입니다. **실데이터** 기반입니다.

- **데이터 출처**: 고캠핑(한국관광공사) 공개 API — 이용허락범위 제한 없음
- **수록**: 김포 3시간 이내 자동차야영장(오토캠핑) **143곳** (파주·연천·포천·강원)

## 실데이터로 되는 것 / 예약처에서 확인할 것

| 항목 | 상태 |
|---|---|
| 캠핑장명·지역·주소·오토 규모 | ✅ 고캠핑 실데이터 |
| 김포 거리·예상 이동시간(좌표 계산)·3시간 필터 | ✅ |
| 계절 추천 (여름 물놀이·계곡 / 겨울 온수·눈 등) | ✅ 테마환경 태그 기반 |
| 시설·환경 태그, 대표 사진, 예약처 링크 | ✅ |
| **가격 · 실시간 자리 · 후기** | ❌ API 미제공 → 예약처/네이버로 연결 |

> 후기 데이터 실제 연동과 실시간 자리 조회(“실시간” 배지)는 백엔드가 필요한 **2단계 과제**입니다.

## 기능

- **추천순 랭킹** — 계절 적합 + 시설 충실도 + 오토 규모 + 아이 좋은 곳 + 온라인예약 − 거리
- **계절 추천 큐레이션** — 조회일 계절에 맞는 곳을 상단에 우선 노출
- **선호 우선** — ☆ 지정 캠핑장을 최상단 고정 (localStorage 저장)
- **아이 좋은 곳 배지** — 놀이터·물놀이장·계곡 등 보유 시 표시
- 상세: 사진 · 시설 태그 · **예약처 이동 / 네이버 후기 / 김포에서 길찾기(카카오맵)**
- 라이트/다크 테마 자동, 모바일 우선 반응형

## 조건 (기본값)

- 지역: 파주 · 연천 · 포천 · 강원 (김포 집 기준 약 3시간 이내)
- 유형: 오토캠핑

## 파일 구조

```
camping-dashboard/
├─ index.html
├─ css/styles.css
├─ js/
│  ├─ config.js     # 계절·지역 설정 (수기)
│  ├─ sites.js      # 캠핑장 실데이터 (자동 생성)
│  ├─ logic.js      # 랭킹·계절·링크 로직
│  └─ app.js        # 상태·렌더링·이벤트
└─ tools/build_data.py   # 고캠핑 API → sites.js 생성기
```

## 데이터 갱신 (실데이터 재생성)

고캠핑 API 키는 **환경변수로만** 사용하며 저장소에 커밋하지 않습니다.

```bash
GOCAMPING_KEY="발급받은_일반인증키" python3 tools/build_data.py
# → js/sites.js 재생성 (김포 3시간 이내 오토캠핑장)
```

API 키 발급: [공공데이터포털](https://www.data.go.kr) → “한국관광공사_고캠핑 정보 조회서비스” 활용신청.

## 실행 / 배포

빌드 과정 없음. `index.html`을 브라우저로 열거나 로컬 서버(`python3 -m http.server`)로 확인.
GitHub Pages 정적 호스팅으로 배포합니다.
